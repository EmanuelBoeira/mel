mel(Mecanical ELements) é um projeto que busca fornecer uma
interface amigável para o projeto de elementos de máquina e,
além disso, realizar os calculos de critérios de falha de 
maneira rápida.  

Este é um projeto acadêmico de desenvimento de un software
para calculos de elementos de máquina para o trabalho de 
conclusão do curso de engenharia mecânica.
